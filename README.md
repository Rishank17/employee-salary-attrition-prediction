# Employee Salary & Attrition Prediction

A Machine Learning project that analyzes HR data to predict employee attrition and understand factors affecting employee retention and compensation.

## Dataset

The dataset contains employee-related information such as:

- Age
- Gender
- Education
- Job Role
- Department
- Monthly Income
- Years at Company
- Job Satisfaction
- Performance Rating
- Work-Life Balance
- Attrition Status

## Objectives

- Analyze employee demographics
- Identify factors contributing to attrition
- Predict employee attrition
- Explore salary patterns across job roles
- Build and evaluate machine learning models

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
